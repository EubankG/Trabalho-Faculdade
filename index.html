Projeto A3

Somos estudantes da Universidade São Judas Tadeu. Estamos cursando Análise de Dados e Big Data e Inteligência Artificial. O projeto desse semestre é realizar uma análise dos dados de algum dataset escolhido, pelos integrantes do grupo que irá realizar o projeto.

Com isso escolhemos o dataset da Steam para analisar os dados. A observação desses dados é muito importante para acompanhar a evolução dos jogos e da tecnologia e como isso influência no preço final do jogo para ser vendido pelas produtoras.

Steam é uma das maiores plataformas de jogos e uma das melhores também, pois, ela dá suporte ao cliente que comprou e ao que desenvolveu e publicou o jogo nessa plataforma. Foi realizado diversos gráficos fazendo algumas análises como os gêneros mais jogados, sistemas operacionais mais utilizados para jogos, quantidade de jogos lançados por ano e os maiores preços dos jogos a cada ano.

Como veremos nos gráficos teve uma evolução muito grande de jogadores e da tecnologias nas últimas décadas, não só esses aspectos mas o preço dos jogos também aumentaram.
