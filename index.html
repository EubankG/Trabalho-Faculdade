Projeto A3

Somos estudantes da Universidade São Judas Tadeu. Estamos cursando Análise de Dados e Big Data e Inteligência Artificial. O projeto desse semestre é realizar uma análise dos dados de algum data set escolhido, pelos integrantes do grupo que ira realizar o projeto.

Com isso escolhemos o data set da Steam para analisar os dados. A observação desses dados é muito importante para acompanhar a evolução dos jogos e da tecnologia e como isso influência no preço final do jogo para ser vendido pelas produtoras.

A steam é uma das maiores plataformas de jogos e uma das melhores também, pois ela dá suporte ao cliente que comprou e ao que desenvolveu e publicou o jogo nessa plataforma. Foi realizado diversos gráficos fazendo algumas análises como os jogos mais jogados de cada gênero, quantos jogadores cada jogo terá no próximo ano e quanto custará cada jogo com o passar do tempo.

Como observado nos gráficos e dataset teve uma evolução muito grande de jogadores e da tecnologias nas últimas décadas, não só esses aspectos mas o preço dos jogos também aumentaram. Vimos que a steam teve um número absurdo de usuários e com isso se torna umas das maiores plataformas de jogos.
